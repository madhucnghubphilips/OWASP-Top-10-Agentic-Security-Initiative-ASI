# OWASP-Top-10-Agentic-Security-Initiative-ASI-
The OWASP Top 10 for Agentic Applications (2026) is an initiative from the OWASP Agentic Security Initiative (ASI) focused on security risks specific to AI agents that can reason, plan, use tools, access systems, communicate with other agents, and take autonomous actions.
